UserStatusService.UserStatusService Class

Extends AbstractService.AbstractService
Defined in: src\routes\userStatusService.ts:56

/user

Declares User status API which provides methods to persist user status and allows user context switching. This functionality is used in cooperation with NFC to demonstrate IVI adaptation when approached by different users using contactless method. Users are identified by cookie iviUser sent from server back to NFC-enabled device.

[GET] /login Login by new user. If no iviUser cookie is sent in request or cookie content is not recognized as registered user it's considered a new user and current status is stored with new identificator. Otherwise current settings are stores for last selected user and user data from cookie are retrieved and setup.
