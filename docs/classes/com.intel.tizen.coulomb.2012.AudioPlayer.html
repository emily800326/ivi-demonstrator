com.intel.tizen.coulomb.2012.AudioPlayer Class

Defined in: themes\com.intel.tizen\coulomb.2012\components\audioPlayer\audioPlayer.js:5

Represents HTML5 audio, that is utilizing Audio Service API to get/set status of currently playing song (tested with wav audio files). Audio resources are passed to the player as an array of songs (model/playlist). Audio player is extended by option to set buttons to control audio playback (play/pause, next, previous, shuffle, repeat), time progress bar, spectrum analyzer, info panel, volume control.
