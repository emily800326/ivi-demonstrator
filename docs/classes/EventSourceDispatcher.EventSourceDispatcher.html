EventSourceDispatcher.EventSourceDispatcher Class

Extends EventEmitter
Defined in: src\eventSourceDispatcher.ts:41

Declares single point of dispatching HTML5 EventSource messages using text/event-stream content type to avoid WebKit limitations. This class works as server-side complementary component to class ApiEndpointBase. WebKit engine currently supports up to 5 parallel EventSource sources. This can be avoided by declaring one common source and dispatching messages based on identificator. To send new message use emit method e.g.:

 eventSourceDispatcher.emit("send", { source: "mysource", message: "Test message" });
