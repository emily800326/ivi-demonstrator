NFC integration

To enable WiFi tethering on IVI unit execute following commands:

  
    cd /usr/lib/conman/test
    ./enable-tethering wifi [SSID] [password] # e.g. ./enable-tethering wifi ivi testing-123

To find out IP address of WiFi interface type 

  
    ifconfig tether

Then on NFC enabled phone connect to WiFi network with SSID / password specified above and program the NFC tag to following actions:

  1. Enable WiFi
  2. Connect to WiFi network
  3. Launch http://[tethering_ip_address]:8088/user/login (e.g. http://192.168.0.1:8088/user/login)

Now use NFC enabled tag with NFC tag to change profiles. Currently Car Indicator profile and user theme are preserved. Information about registered devices is not stored persistently meaning that new launch of Engine server will lose all registered devices.
